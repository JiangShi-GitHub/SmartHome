# SmartHome
一个基于Arm-6816开发板的智能家具管理系统

## 项目优势

>1.完成全部基本功能且无任何bug；
>
>2.全局的报警设计；
>
>3.合理的框架设计和严谨的线程管理；
>
>4.动态连续的RGB变换；
>
>5.歌曲音量调节、基于双链表的切歌；
>
>6.界面简洁明了美观，设计创新；
